I couldn't quite get the Node options working. nodejs/npm install can be skipped if you don't need those.

on ubuntu 16.04:
apt-get install nodejs npm python3-venv
ln -s /usr/bin/nodejs /usr/bin/node

Follow the instructions in the benchmark repo above to set up a virtualenv. 


# pull latest Go. You can always get it from golang.org/dl/ if this doesn't work.
wget https://redirector.gvt1.com/edgedl/go/go1.9.2.linux-amd64.tar.gz
tar -C /usr/local -xzf go1.9.2.linux-amd64.tar.gz
export PATH=$PATH:/usr/local/go/bin

apt-get install -y postgresql

edit:
/etc/postgresql/*/main/pg_hba.conf

set this to 'trust' instead of 'peer':

# Database administrative login by Unix domain socket
local   all             postgres                                peer

change this to trust too instead of md5:
host    all             all             127.0.0.1/32            md5

sudo service postgresql reload


Edit the pgbench file to remove node benchmarks from the following list:
BENCHMARKS = [ ... 

Run bench. Takes ~20 min or so. 
./pgbench --pguser postgres --pghost 127.0.0.1 -H results.html -J results.json
